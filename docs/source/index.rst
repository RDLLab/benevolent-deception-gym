Welcome to Benevolent Deception Gym's documentation!
====================================================

Benevolent Deception Gym (BDGym) is a collection of multi-agent `OpenAI gym <https://github.com/openai/gym>`_ environments that incorporates the potential for deception for the benefit of the decieved (benevolent deception).

How should I cite BDGym?
------------------------

Please cite BDGym in your publications if you use it in your research. Here is an example BibTeX entry:

.. code-block:: bash

    @misc{schwartz2021bdgym,
    title={BDGym: Benevolent Deception Gym},
    author={Schwartz, Jonathon and Kurniawatti, Hanna},
    year={2021},
    publisher = {GitHub},
    journal = {GitHub repository},
    howpublished = {\url{https://github.com/RDLLab/benevolent-deception-gym}},
    }


If you have used the `Driver-Assistant` BDGym environment please also cite the `Highway-Env <https://github.com/eleurent/highway-env>`_ environment (recommended citation is available in the `Highway-Env` github README).
